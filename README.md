# PingUp — Social Media App

PingUp is a full-stack social media platform that allows users to connect, share posts, interact through likes and comments, and engage with real-time content. The app is designed with scalability, performance, and a modern UI/UX in mind.

---

## Features

### Authentication & Users

- Secure user authentication (powered by Clerk)
- User profiles with editable information
- Follow / connect with other users

### Posts

- Create posts with text and images
- View a global and personalized feed
- Delete and manage your posts

### Interactions

- Like / unlike posts
- Comment on posts
- Real-time engagement updates

### Media Uploads

- Image uploads via Multer / cloud storage
- Support for multiple images per post

### Stories

- Temporary story feature (auto-expiring)
- View stories from connections

---

## Tech Stack

### Frontend

- React
- Tailwind CSS
- Redux Toolkit (state management)
- React Router

### Backend

- Node.js + Express
- MongoDB + Mongoose
- Inngest (background jobs / automation)
- Multer (file uploads)

- Clerk
